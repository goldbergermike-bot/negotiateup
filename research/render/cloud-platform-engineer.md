#### Negotiation DNA — Why This Role Commands a Premium at Render

Render is a private startup headquartered in San Francisco building the modern successor to Heroku. The platform provides web services, static sites, background workers, cron jobs, managed PostgreSQL databases, Redis, and private networking — all deployed directly from Git with automatic builds and deploys.

Cloud Platform Engineers are the signature technical role at Render — they build the actual cloud infrastructure that the product runs on. This includes container orchestration, network overlay systems, build pipelines, resource scheduling, and the control plane that turns git pushes into running services. At a small startup, these engineers define the architectural foundations of the entire product.

This role commands a premium because cloud platform engineering talent is scarce and the work is foundational to Render's competitive position against Heroku, Fly.io, Vercel, Railway, and the major cloud providers.
