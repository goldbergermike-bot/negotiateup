#### Negotiation DNA

Security Engineering at Tastytrade carries stakes that most application security roles do not. You are securing a platform where breaches translate directly into stolen financial assets, where unauthorized access to trading accounts can result in fraudulent derivatives trades with leveraged losses, and where regulatory penalties from the SEC, CFTC, FINRA, and FCA can reach hundreds of millions of dollars. The security posture of a derivatives trading platform is not an abstract concern — it is existential, and security engineers who understand the specific threat landscape of options trading platforms are extraordinarily valuable.

Tastytrade's derivatives-native platform creates security challenges that generic fintech security engineers rarely encounter. Multi-leg options orders must be authenticated and authorized with atomic guarantees — a partially-executed multi-leg order due to a security timeout creates financial risk. Real-time market data feeds must be secured against manipulation without introducing latency that degrades trading performance. Greeks computation and risk management systems contain proprietary algorithms that represent core intellectual property. Each of these derivatives-specific security challenges requires an engineer who understands both cybersecurity and options trading mechanics.

The IG Group acquisition has dramatically expanded the security surface. You are now responsible for security across a global platform operating under multiple regulatory regimes — SEC/CFTC in the US, FCA in the UK, and various international regulators across IG Group's global footprint. Cross-border data flows, multi-jurisdictional compliance requirements, and the integration of two previously independent security architectures create a security engineering challenge of exceptional complexity.

#### Options Alpha — The Derivatives-Native Premium

As a Security Engineer at Tastytrade, the Derivatives-Native premium reflects your understanding of the specific security requirements and threat models of derivatives trading platforms — threats that differ materially from those facing generic fintech applications.

- **The Derivatives-Native Bonus (8-15% above standard fintech):** Security Engineers with trading platform and financial regulatory security expertise command a Derivatives-Native bonus of $15K-$42K above standard security engineer compensation at comparable fintech firms. This premium reflects your understanding of derivatives-specific threat models (multi-leg order manipulation, options pricing data tampering, Greeks computation exploitation), financial regulatory security requirements (SEC Rule 15c3-5, CFTC cybersecurity guidelines, FCA operational resilience), and the unique security architecture requirements of real-time trading systems. At the security engineer level, this typically manifests as a $14K-$24K base uplift and a $7K-$18K annual bonus uplift.

- **High-Probability Trading Focus:** Tastytrade's high-probability trading features — probability of profit calculations, implied volatility analytics, Greeks dashboards — rely on data integrity that must be guaranteed by security systems. If a threat actor can manipulate the implied volatility data feeding Tastytrade's probability calculations, they can mislead millions of traders into making suboptimal decisions. Security engineers who understand the derivatives domain well enough to identify these domain-specific attack vectors — and to design security controls that protect against them without degrading platform performance — command the Derivatives-Native premium.

- **IG Group Global Integration:** Security Engineers driving the Tastytrade-IG Group security integration face the challenge of harmonizing security architectures across two organizations, two regulatory frameworks, and multiple geographies. You are defining the security model for cross-border data flows (US-UK), multi-jurisdictional identity and access management, and unified threat detection across platforms that handle different asset classes (options, FX, CFDs). Each regulatory regime has specific cybersecurity requirements — SEC/CFTC in the US, FCA's operational resilience framework in the UK — and your security architecture must satisfy all simultaneously.

- **tastylive Media Network Synergy:** Tastytrade's unique combination of trading platform and live financial media network (tastylive) creates a security surface that spans both financial platform security and media infrastructure security. You must secure systems where live financial content production interacts with real-time trading platform data — ensuring that market-sensitive information is handled securely across both the trading and media components. This dual-security-domain challenge is unique to Tastytrade.
