#### The Ecosystem Platform Engineer's Scope — What You Own

Unlike other engineering roles that own a product vertical or a technical domain, the Ecosystem Platform Engineer owns the horizontal platform layer that every vertical depends on. This scope is why the role commands SIGNATURE ROLE compensation:

**1. Unified API Gateway and Service Mesh**
You design and build the API gateway and service mesh that enables Robinhood's six product verticals to communicate with each other and with external partners. This is the nervous system of the ecosystem — handling authentication, rate limiting, service discovery, load balancing, circuit breaking, and cross-product routing for billions of API calls per day. Your gateway design determines whether the ecosystem feels like one platform or six disconnected APIs.

**2. Cross-Product Event Bus and Data Contracts**
You architect the event-driven infrastructure that enables real-time cross-product data flows — trading activity triggering retirement portfolio rebalancing, crypto transactions updating cash management balances, credit card spending informing investment recommendations. You define the event schemas and data contracts that ensure six product teams can publish and consume events reliably without tight coupling.

**3. Ecosystem Identity, Authorization, and Entitlements**
You own the identity framework that gives each user a single, consistent identity across all product verticals — with granular authorization and entitlement management that controls which features, data, and services each user can access based on their account type, regulatory status, and subscription tier (including Robinhood Gold).

**4. Shared Ledger, Reconciliation, and Compliance Engine**
You build the shared financial ledger and reconciliation system that tracks all financial transactions across the ecosystem — ensuring that every dollar, crypto token, and credit transaction is accurately recorded, reconciled, and reportable for SEC, FINRA, banking, and tax compliance across multiple product verticals simultaneously.

**5. Institutional Data Platform**
You architect the platform that transforms Robinhood's internal ecosystem data into institutional-grade data products — market data feeds, research analytics, aggregated trading signals, and compliance data sets that the Rothera JV will sell to institutional clients. This is a new revenue line built directly on the platform you create.

**6. Multi-Vertical Observability and Reliability**
You own the ecosystem-wide observability platform — the monitoring, alerting, tracing, and reliability dashboards that give Robinhood's engineering leadership visibility into cross-product system health. Your observability architecture determines whether incidents in one vertical cascade across the ecosystem or are contained.
